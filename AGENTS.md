# Koya Business OS for Codex

Use this repository as a Koya / FamilyTreeOki dashboard and app-generation base.

## Rules

- Read the existing code before editing.
- Treat FamilyTreeOki as a US military family life-event support platform, not only an ultrasound service.
- Use the frame: US dollar-income demand x Okinawa yen-cost operation x high-emotion life event.
- Never invent current numbers.
- Keep observed, estimated, mock, fallback, unknown, and blocked_or_failed separate.
- Put external data behind source adapters.
- Do not hardcode API keys or secrets.
- Flag TRICARE, medical advertising, qualification, childcare, and babysitting wording for human review.

## Expected Stack

- React / Vite
- Tailwind CSS
- Recharts
- Future source adapters for FRED, BLS, FRB, DFAS, DTMO/OHA, TRICARE, internal bookings, sales, and reviews
